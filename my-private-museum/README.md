# My Private Museum website

Static, dependency-free GitHub Pages files for:

- `/my-private-museum/`
- `/my-private-museum/privacy/`
- `/my-private-museum/support/`
- `/my-private-museum/legal/`

## Before publishing

1. Replace every placeholder in `legal/index.html` with a current postal address.
2. Remove the publication-hold notice from `legal/index.html`.
3. Remove `<meta name="robots" content="noindex">` from `legal/index.html`.
4. Confirm that `Ali Babi` is the publisher name used in the app stores.
5. Confirm that `ababi@gmx.de` is ready to be publicly visible.
6. Review the policy after every material change to app data handling or dependencies.

The site intentionally contains no JavaScript, analytics, advertising, remote fonts,
remote images, contact forms, or app-download tracking.
